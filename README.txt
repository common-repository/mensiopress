e-Commerce System For Wordpress

== Description ==
Mensiopress is an e-commerce web application, available as a plug-in for Wordpress. It aims to cover the needs of most e-commerce business, while attempting a deeper and more effective approach than any similar single plugin available.

Main Features:

1. Extensive Product Catalog Maintenance Utilities
2. Multi-Language Support
3. Block-Based Page Builder
4. Selectively Assigned Administration Permissions
5. Multi-Currency Support
6. Custom Shipping Methods
7. Basic Reporting Tools
8. Unique User-Interface

== Installation ==
1. Allow the plugin to automatically install.
2. Select the "Mensiopress" icon from the sidebar.
3. Wait until the initial setup is completed.
